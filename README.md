# Laravel フォーム投稿機能 + バリデーションテスト

このプロジェクトは、Laravelを使って**フォーム投稿機能**を実装し、**バリデーションのテスト**を行うためのサンプルプロジェクトです。

## 🚀 機能一覧
- フォームからの投稿（タイトル & 内容）
- 投稿データのバリデーション
- 投稿一覧の表示
- 投稿成功時のリダイレクト
- **ユニットテスト & Featureテスト付き**

## 🛠 使用技術
- **PHP** 8.x
- **Laravel** 10.x
- **MySQL** (または SQLite)
- **PHPUnit**（Laravelのテスト機能）
